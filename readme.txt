Сервер работает с БД, где предварительно необходимо создать БД "NettyServer".
Создание таблицы в этой БД необходимо произвести запросом из прилагаемого файла(query.txt).
Приложение запускается из класса "ServerNetty".
В данном проекте прилагается скриншот теста обращений и страницы /status.
В задании использована библиотека Netty 4.0.1.Final.
